Macro Definition Documentation

#define MAX_MSGQ_SIZE   (uint8_t)0x8u

Maximum size of one single message queue.

Note
MUST BE A POWER OF TWO!
Warning
If the amount of data elements in the message queue exceeds the MAX_MSGQ_SIZE the oldest element is dropped!
#define MAX_QUEUE_DATA_ELEMENT_SIZE   (uint8_t)0x10u

Defines the amount of data one queue can hold.

Warning
CHOOSE WITH CAUTION! MEMORY IS EXPENSIVE!
